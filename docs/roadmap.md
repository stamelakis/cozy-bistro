- Vite, TypeScript, and Phaser setup
- Top-down restaurant room
- Grid rendering
- Place, move, and remove furniture
- Placeholder geometric art

Status: implemented.

## Stage 2: Money and Decoration Score

- Furniture costs money
- Removal refund
- Decoration score from comfort and style
- Attractiveness estimate

Status: partially implemented and testable.

- Interactive cooking menu
- Recipe queue

- Station requirements
- Produced servings

- Spawn customers based on attractiveness, seats, menu quality, and service speed
- Seat customers
- Generate orders
- Waiting and patience timers

## Stage 5: Reputation and Day Loop

- Day timer
- End-of-day summary
- Daily customer count
- Satisfaction changes
- Reputation changes

## Stage 6: Save and Load

- Expand localStorage save coverage
- Add explicit load/new game controls
- Add save version migration path

## Stage 7: Cozy Polish

- Replace placeholder geometry with original art
- Add subtle room animation
- Improve UI clarity
- Add audio and feedback
- Tune economy and customer pacing
